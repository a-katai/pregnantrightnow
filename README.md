# Pregnant Right Now - Landing Page

This Next.js application serves a landing page for `pregnantrightnow.com` with information about the EP release party and a link to RSVP.

## Setup

1. Install dependencies:
```bash
bun install
```

2. Run development server:
```bash
bun dev
```

## Deployment

Deploy to Vercel (recommended) or any platform that supports Next.js:

1. Connect your repository to Vercel
2. Add `pregnantrightnow.com` as a custom domain
3. Deploy

The landing page features a beautiful design matching the RSVP page style, with information about the February 20th EP release party and a link to RSVP at `chariahs.world/rsvp`.
